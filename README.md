## RFM ANALYSIS FOR E-COMMERCE DATA
## Customer Segmentation using RFM Analysis

## Project Overview

This project focuses on customer segmentation using RFM (Recency, Frequency, Monetary) analysis to identify customer purchasing behavior and business value. 
The analysis was performed using SQL in Google BigQuery, followed by data visualization in Tableau to generate actionable business insights.

The project helps businesses:
- Identify high-value customers
- Detect inactive or at-risk customers
- Improve customer retention strategies
- Optimize targeted marketing campaigns

---

## Tools & Technologies
- Google BigQuery
- SQL
- Tableau
- Data Visualization

---

## Project Workflow
1. Data Cleaning & Preparation  
2. Customer-Level Aggregation  
3. RFM Metric Calculation  
4. RFM Score Generation  
5. Customer Segmentation  
6. Data Visualization & Insights

---

## Customer Segments Identified
- Champions
- Loyal Customers
- Potential Loyalists
- At Risk Customers
- Lost Customers

---

## Key Business Insights
- High-frequency customers contributed significantly to total revenue.
- Several customers showed high monetary value but low recent activity, indicating re-engagement opportunities.
- Customer segmentation enables more personalized marketing strategies.
